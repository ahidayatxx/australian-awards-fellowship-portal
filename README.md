# Australia Awards Fellowship - Value-Based Digital Health Transformation Indonesia

## Project Portal

This repository contains the project portal for the Australia Awards Fellowship program focused on value-based digital health transformation in Indonesia.

## Access the Portal

**Live Project Portal:** [https://ahidayatxx.github.io/australian-awards-fellowship-portal/project_portal.html](https://ahidayatxx.github.io/australian-awards-fellowship-portal/project_portal.html)

## About the Program

- **Timeline:** November 8, 2025 - February 21, 2026 (15 weeks)
- **Participants:** 9 fellows (5 government from TWG SATUSEHAT, 4 academic from eastern Indonesia universities)
- **Objective:** Implement value-based digital health transformation through intensive fellowship program

## Project Structure

- `project_portal.html` - Interactive project dashboard with timeline, deliverables, team information, and risk assessment
- Supporting documentation and resources for program implementation
